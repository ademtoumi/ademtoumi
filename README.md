## Featured Projects

### [Log-Based Anomaly Detection](https://github.com/ademtoumi/log-anomaly-detection) — Master's Thesis
> **Python · Kafka · HuggingFace Spaces · PostgreSQL · Streamlit · SHAP/LIME**

End-to-end research and deployment system for log-based anomaly detection in distributed computing environments. Benchmarks 10+ ML/DL models (SVM, Random Forest, BiLSTM, CNN-BiLSTM, DeepLog, Hierarchical Attention) across three real-world log datasets (HDFS, BGL, Spirit). Culminates in a production-grade real-time streaming pipeline: Kafka → deployed HuggingFace Spaces inference → PostgreSQL storage → Streamlit dashboard with XAI explanations.

---

### [UAV Object Detection & Tracking](https://github.com/ademtoumi/UAV-Object-Detection-Tracking) — Engineering Project
> **Python · YOLOv11 · OpenCV · Raspberry Pi · Tkinter**

Hardware-software UAV surveillance system for defense and security applications. My contribution: training and deploying YOLOv11s on the VisDrone dataset, implementing HyperDet CNN, and building the data pipeline for aerial object detection (~90% mAP@0.5, 40 FPS on GPU). Integrated with a Raspberry Pi 5 + Intel Movidius NCS 2 edge system and a full Tkinter Ground Control Station.

### [Hajj & Umrah Platform](https://github.com/ademtoumi/hajj-platform-frontend) — Full-Stack Web Application
> **Next.js 14 · TypeScript · Tailwind CSS · Socket.io · TanStack Query**

Trilingual (Arabic/English/French) civic-tech platform for managing Hajj pilgrimage registration in Algeria. Includes a user-facing app (hotel browsing, reservations, payments, lottery draw system, examination tracking) and a separate admin dashboard (applicant management, flight scheduling, Wilaya/Commune data, real-time updates via Socket.io, Recharts analytics).

---

### [Malicious URL Detection](https://github.com/ademtoumi/Malicious-URL-Detection) — ML & Cybersecurity
> **Python · Scikit-learn · XGBoost · TensorFlow · Jupyter**

Comparative analysis of ML/DL models for URL threat classification (benign, phishing, malware, defacement) on 651K URLs. Implements TF-IDF character n-gram features (3–5 grams) combined with 15+ handcrafted URL features. Best result: Random Forest at **96.8% F1-score**. Co-authored with Ahmed Saadi.

---

### [Paraphrase Detection](https://github.com/ademtoumi/Paraphrase-Detection) — Deep Learning & NLP
> **Python · PyTorch · HuggingFace · spaCy · NLTK**

Siamese BiLSTM (3 layers, 512 hidden) with 4-head multi-head attention for semantic similarity detection. Trained on PAWS + Quora Question Pairs (~450K sentence pairs) with WordNet-based augmentation and Focal Loss for class imbalance. Co-authored with Ahmed Saadi.

---

### [KidLink](https://github.com/ademtoumi/KidLink-App) — Mobile Application (PFE)
> **Flutter · Dart · Firebase · Provider**

Cross-platform parent-teacher communication app for primary schools. My contribution: Flutter front-end development including UI screens, Firebase integration, and push notifications. Features role-based access (parent/teacher/admin), real-time FCM notifications, and student progress dashboards. 6-person team project, ESI-SBA 2022/2023.
