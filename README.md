## Featured Projects

### [Log-Based Anomaly Detection in Distributed Systems](https://github.com/ademtoumi/log-anomaly-detection)
*Engineering & Master's Thesis — ESI-SBA, July 2026*

Systematic evaluation of eleven ML and DL models (SVM, Random Forest, Isolation Forest, BiLSTM, CNN-BiLSTM, DeepLog, LSTM-AE, HierAttn, and others) for log-based anomaly detection on HDFS, BGL, and Spirit datasets. Includes a fully operational real-time streaming pipeline: Apache Kafka → HuggingFace Spaces inference → PostgreSQL → Streamlit dashboard with SHAP and LIME explainability.

`Python` `PyTorch` `TensorFlow` `Scikit-learn` `Apache Kafka` `Streamlit` `SHAP` `LIME` `HuggingFace`

📄 [Engineering Thesis](https://github.com/ademtoumi/log-anomaly-detection/blob/main/Engineer_Thesis.pdf) · [Master's Thesis](https://github.com/ademtoumi/log-anomaly-detection/blob/main/Master_Thesis.pdf)

### [Object Detection and Tracking for UAV Applications](https://github.com/ademtoumi/UAV-Object-Detection-Tracking)
*Academic Internship — URD Laboratory, 2024–2025*

Real-time aerial surveillance system integrating YOLOv11s object detection (trained on VisDrone and military vehicle datasets) with DeepSort and CSRT tracking, deployed on a Raspberry Pi 5 with Intel Movidius NCS 2. Includes a Tkinter-based Ground Control Station with live telemetry and gimbal control over UDP.

`Python` `PyTorch` `Ultralytics YOLOv11` `OpenCV` `DeepSort` `Raspberry Pi`

---

### [HajjPortal — Web Platform for Hajj Management](https://github.com/ademtoumi/hajj-platform-frontend)
*Academic Project — ESI-SBA, 2023–2024*

Trilingual (Arabic, English, French) web platform for Hajj pilgrimage management, covering pilgrim registration, hotel reservation, lottery draw administration, and journey tracking. Built with Next.js 14 App Router and Socket.io real-time updates. Comprises a [user-facing application](https://github.com/ademtoumi/hajj-platform-frontend) and an [administrative dashboard](https://github.com/ademtoumi/hajj-platform-admin).

`TypeScript` `Next.js` `React` `Tailwind CSS` `Socket.io` `TanStack Query`

---

### [Machine Learning-Based URL Classification](https://github.com/ademtoumi/Malicious-URL-Detection)
*Machine Learning Project — ESI-SBA, 2024–2025*

Comparative study of Logistic Regression, Random Forest, XGBoost, and LSTM for four-class URL classification on 651,191 URLs. Combines TF-IDF character n-gram features with handcrafted structural URL descriptors. Best result: Random Forest at 96.8% F1-score.

`Python` `Scikit-learn` `XGBoost` `TensorFlow` `Pandas`

📄 [Project Report](https://github.com/ademtoumi/Malicious-URL-Detection/blob/main/report.pdf)

---

### [Paraphrase Detection with Siamese Neural Networks](https://github.com/ademtoumi/Paraphrase-Detection)
*NLP & Deep Learning Project — ESI-SBA, 2024–2025*

Siamese BiLSTM (3 layers, 512 hidden units) with 4-head multi-head attention for semantic similarity detection, trained on PAWS and Quora Question Pairs (~450K sentence pairs) using WordNet augmentation and Focal Loss.

`Python` `PyTorch` `HuggingFace` `spaCy` `NLTK`
