## Projects

**[Log-Based Anomaly Detection in Distributed Systems](https://github.com/ademtoumi/log-anomaly-detection)**  
*Engineering & Master's Thesis — ESI-SBA, July 2026*  
Systematic evaluation of eleven machine learning and deep learning models for log-based anomaly detection on HDFS, BGL, and Spirit datasets. Includes a fully operational real-time streaming pipeline built on Apache Kafka, HuggingFace Spaces inference, PostgreSQL, and Streamlit with SHAP and LIME explainability.

**[Object Detection and Tracking for UAV Applications](https://github.com/ademtoumi/UAV-Object-Detection-Tracking)**  
*Academic Internship — URD Laboratory, 2024–2025*  
Real-time aerial surveillance system integrating YOLOv11-based object detection with DeepSort tracking, deployed on a Raspberry Pi 5 with Intel Movidius NCS 2 acceleration and a PC-based Ground Control Station with gimbal control.

**[HajjPortal — Web Platform for Hajj Management](https://github.com/ademtoumi/hajj-platform-frontend)**  
*Academic Project — ESI-SBA, 2023–2024*  
A trilingual (Arabic, English, French) web-based platform for Hajj pilgrimage management, supporting pilgrim registration, service allocation, lottery administration, and operational workflows. Built with Next.js 14, TypeScript, and Socket.io.

**[Machine Learning-Based URL Classification](https://github.com/ademtoumi/Malicious-URL-Detection)**  
*Machine Learning Project — ESI-SBA, 2024–2025*  
Comparative study of Logistic Regression, Random Forest, XGBoost, and LSTM for four-class URL classification evaluated on 651,191 URLs, with TF-IDF character n-gram features combined with handcrafted structural URL features.

**[Paraphrase Detection with Siamese Neural Networks](https://github.com/ademtoumi/Paraphrase-Detection)**  
*NLP & Deep Learning Project — ESI-SBA, 2024–2025*  
Siamese BiLSTM with multi-head attention for semantic similarity detection, trained on PAWS and Quora Question Pairs using WordNet-based augmentation and Focal Loss.

**[KidLink — School Communication Platform](https://github.com/ademtoumi/KidLink-App)**  
*Engineering Final Project (PFE) — ESI-SBA, 2022–2023*  
Cross-platform mobile application for parent-teacher communication in primary schools, developed with Flutter and Firebase. Team project; contribution: Flutter front-end development.
